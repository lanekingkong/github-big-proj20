# blog-writer

## Description
高质量博客文章生成——SEO优化、原创性增强、多平台格式适配。

## Usage
输入主题和大纲，生成一篇结构清晰、SEO友好的博客文章。

## Instructions

1. 根据主题生成文章大纲（引言-正文-结论）
2. 每段具有明确的主题句和支撑论据
3. 避免AI高频用语（如"在当今时代"、"赋能"等）
4. 自动添加SEO元描述和关键词
5. 生成3-5个备选标题
6. 输出Markdown格式，支持多平台直接发布
7. 如果指定风格预设（professional/creative/academic），按预设调整语气
