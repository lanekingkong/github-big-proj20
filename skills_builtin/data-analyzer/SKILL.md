# data-analyzer

## Description
智能数据分析——自动识别数据结构、选择分析方法、生成可视化报告。

## Instructions

1. 读取数据源（CSV/JSON/SQL/API）
2. 自动识别数据类型和特征
3. 执行探索性数据分析（EDA）：
   - 缺失值检测
   - 异常值识别
   - 分布分析
   - 相关性矩阵
4. 根据数据特征推荐分析方法（回归/分类/聚类/时间序列）
5. 生成Markdown分析报告含可视化建议
6. 对敏感数据列自动脱敏处理
